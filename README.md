# QuizGoFront
QuizGo

QuizGo es una emocionante aplicación móvil de cuestionarios desarrollada con Flutter que permite a los usuarios poner a prueba sus conocimientos en diversas categorías. ¡Desafía a tus amigos y aprende algo nuevo!
✨ Características Principales

    Múltiples Categorías: Una variedad de temas de cuestionarios para elegir (ej. Ciencia, Historia, Deportes, etc.).

    Modo Desafío: Opciones para cronometrar las respuestas y aumentar la emoción.

    Interfaz de Usuario Intuitiva: Diseño limpio y fácil de usar, desarrollado con la rica biblioteca de widgets de Flutter.

    Puntuación y Progreso: Seguimiento del rendimiento del usuario y almacenamiento de puntuaciones más altas.
